# Contributing Guide

**Purpose**: Task management and workflow for revenue sprint
**Lifecycle**: Stable (changes rarely)

## Overview

This is a solo sprint project focused on revenue generation. The workflow is optimized for:
- Daily task tracking
- Weekly metric reporting
- Rapid iteration based on feedback
- Minimal overhead (no elaborate processes)

---

## Task Management Workflow

### Daily Workflow

**Morning (30 minutes):**
1. Review STATUS.md â†’ Today's tasks
2. Check pipeline status (any responses overnight?)
3. Prioritize top 3 tasks for the day
4. Time-block calendar

**During day:**
- Execute tasks (Upwork bids, calls, website work)
- Track time spent per track (Freelance/B2B/Productized)
- Log important conversations/learnings

**Evening (15 minutes):**
1. Update STATUS.md with completed tasks
2. Log any new leads/opportunities
3. Note blockers or challenges
4. Prepare tomorrow's priorities

---

### Weekly Workflow

**Friday Review (1 hour):**
1. Calculate week's metrics
2. Update STATUS.md "Completed This Week" section
3. Record "What worked" and "What didn't work"
4. Plan next week's targets
5. Update CHANGELOG.md if major milestone hit

**Weekend (Optional):**
- Reflect on strategy adjustments
- Prepare materials for next week
- Research new opportunities
- Content creation (LinkedIn, case studies)

## Definition of Done

### Website Deployment (arcforge.au)

**Requirements:**
- [ ] Single-page site with all service offerings
- [ ] Pricing clearly displayed
- [ ] Portfolio section (3 projects minimum)
- [ ] Contact form functional
- [ ] Mobile-responsive
- [ ] SSL certificate active
- [ ] DNS propagated (arcforge.au resolves)
- [ ] Google Analytics installed (optional)

**Deliverable**: Live site at https://arcforge.au

---

### Upwork Profile

**Requirements:**
- [ ] Profile approved by Upwork
- [ ] Professional headline
- [ ] 300+ word overview
- [ ] Portfolio items added (3+)
- [ ] Skills listed (10+)
- [ ] Hourly rate set ($80-100)
- [ ] Profile photo uploaded

**Deliverable**: Active Upwork profile receiving job notifications

---

### Upwork Bid

**Requirements:**
- [ ] Read full job description
- [ ] Customized proposal (not template)
- [ ] Address client's specific pain point
- [ ] Propose clear solution
- [ ] State timeline and price
- [ ] Include relevant portfolio link
- [ ] Professional but personable tone
- [ ] Submitted within 2 hours of job posting

**Deliverable**: Bid submitted with response tracking

---

### Local Business Outreach (Email)

**Requirements:**
- [ ] Personalized greeting (use their name/company)
- [ ] Mention their specific industry
- [ ] State clear value proposition
- [ ] Include credibility marker (case study/portfolio)
- [ ] Simple call-to-action (15-min call)
- [ ] Professional signature with website link
- [ ] Follow-up scheduled (3 days if no response)

**Deliverable**: Email sent and logged in lead tracker

---

### Discovery Call/Meeting

**Requirements:**
- [ ] Calendar invite sent
- [ ] Agenda prepared
- [ ] Research company beforehand
- [ ] Questions prepared (pain points, budget, timeline)
- [ ] Portfolio/examples ready to show
- [ ] Note-taking system ready
- [ ] Follow-up email template ready

**Deliverable**: Notes documented, next steps defined

---

### Proposal Sent

**Requirements:**
- [ ] Scope clearly defined
- [ ] Timeline specified
- [ ] Pricing breakdown included
- [ ] Payment terms stated (50% upfront, 50% on completion)
- [ ] What's included vs not included
- [ ] Support terms (if applicable)
- [ ] Validity period (14 days typical)
- [ ] Easy acceptance method (reply, DocuSign, etc.)

**Deliverable**: Proposal sent, follow-up scheduled (3 days)

---

### Deal Closed

**Requirements:**
- [ ] Contract signed or agreement confirmed
- [ ] Deposit/payment received (50% for custom work)
- [ ] Project kickoff scheduled
- [ ] Scope documented and shared
- [ ] Revenue logged in metrics
- [ ] Invoice sent (if not paid upfront)
- [ ] Celebratory update to CHANGELOG.md!

**Deliverable**: Cash in bank, work scheduled
